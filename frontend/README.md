# DietApp Frontend

Frontend minimale in React sviluppato con Vite, Tailwind CSS e [daisyUI](https://daisyui.com/).

## Comandi principali

```bash
npm install       # installa le dipendenze
npm run dev       # avvia il server di sviluppo
npm run build     # crea i file di produzione
```
